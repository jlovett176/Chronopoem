Chronopoem is a generative clock transposing time information into an unstable, 8-line literary environment. 

Operating on a 360-second cycle, every six minutes the system triggers new formal constraints (drift, jitter, erasure) affecting text physics. A combinatorial engine samples from four "word vaults" (80 fragments) to create 160,000+ variations. As verses fade in, previous iterations are archived in a sidebar Manuscript Log.

I collaborated with Gemini to architect the logic, refine the high-contrast aesthetic, and troubleshoot rendering for a seamless, autonomous browser experience.
